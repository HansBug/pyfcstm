Reference map
=============

Use this page to choose an exact-facts reference. The global navigation lists
each reference page directly and keeps the generated API documentation last.

Promise
-------

This area is for exact facts: commands, DSL forms, report fields, option tables,
template configuration, and the generated API map.

Non-goals
---------

It is not a guided tutorial. Use tutorials and how-to guides when you need a
learning path or a task walkthrough.
