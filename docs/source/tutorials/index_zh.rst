教程
====

当你需要引导式首次成功路径时，请阅读这些页面。教程应当短小、有顺序、运行真实命令，并在细节超过学习路径范围时指向任务指南或参考页。

开始之前
--------

先使用 :doc:`/how_to/installation/index_zh` 安装 pyfcstm，然后按照下面的教程顺序阅读。

推荐路径
--------

1. 按 :doc:`quick_start/index_zh` 跑通端到端流程。
2. 在 :doc:`dsl/index_zh` 中学习第一个 DSL 模型。
3. 使用 :doc:`simulation/index_zh` 执行模型。
4. 使用 :doc:`inspect/index_zh` 检查模型。
5. 使用 :doc:`generation/index_zh` 生成 runtime。
6. 使用 :doc:`visualization/index_zh` 导出图。
