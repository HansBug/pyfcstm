How-to roadmap
==============

Use this page to choose a concrete task guide. The global navigation lists each
guide directly; this roadmap explains which page to open for each job.

Promise
-------

This area is for task-oriented guides: start with a concrete job, follow the
steps, and finish with an observable result.

Non-goals
---------

It is not a parameter encyclopedia, a language reference, or a long design
explanation.

Guide map
---------

* :doc:`installation/index` — install pyfcstm and verify the command-line tools.
* :doc:`cli_workflows/index` — run common CLI workflows.
* :doc:`dsl/index` — perform concrete DSL authoring tasks.
* :doc:`simulation/index` — run interactive and batch simulation tasks.
* :doc:`inspect/index` — produce and use inspect reports.
* :doc:`generation/index` — generate code with built-in templates.
* :doc:`visualization/index` — export PlantUML diagrams and rendered images.
* :doc:`templates/index` — maintain or author templates.
* :doc:`grammar_editor/index` — update grammar, highlighting, and editor assets.
