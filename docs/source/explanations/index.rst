Explanation map
===============

Use this page to choose background reading. The global navigation lists each
explanation topic directly; this map explains how the topics fit together.

Promise
-------

This area explains semantics, architecture, boundaries, and trade-offs so
readers understand why pyfcstm behaves as it does.

Non-goals
---------

It is not a step-by-step task guide and should not become the only source for
exact CLI or DSL facts.
