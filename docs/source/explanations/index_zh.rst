解释地图
========

本页用于选择背景阅读。全站左侧导航会直接列出每个解释主题；本地图说明这些主题之间的关系。

本区主承诺
----------

本区解释语义、架构、边界和取舍，帮助读者理解 pyfcstm 为什么这样工作。

本区非目标
----------

本区不是逐步操作指南，也不应该成为 CLI 或 DSL 精确事实的唯一来源。
