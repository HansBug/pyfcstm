任务指南路线图
==============

本页用于选择具体任务指南。全站左侧导航会直接列出每个指南；本路线图说明每类任务应该打开哪个页面。

本区主承诺
----------

本区用于任务导向指南：从一个具体任务出发，按步骤完成，并得到可观察结果。

本区非目标
----------

本区不是参数大全、语言参考，也不是长篇设计解释。

指南地图
--------

* :doc:`installation/index_zh` — 安装 pyfcstm 并验证命令行工具。
* :doc:`cli_workflows/index_zh` — 运行常见 CLI 工作流。
* :doc:`dsl/index_zh` — 完成具体 DSL 编写任务。
* :doc:`simulation/index_zh` — 运行交互式和批处理仿真任务。
* :doc:`inspect/index_zh` — 生成并使用 inspect 报告。
* :doc:`generation/index_zh` — 使用内置模板生成代码。
* :doc:`visualization/index_zh` — 导出 PlantUML 图和渲染图片。
* :doc:`templates/index_zh` — 维护或编写模板。
* :doc:`grammar_editor/index_zh` — 更新 grammar、highlighting 和 editor assets。
